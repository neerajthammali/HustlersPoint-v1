Images for the site

- Place uploaded images here.
- Recommended structure:
  - `authors/` — author avatars (e.g. `neeraj-thammali.jpg`), square, ~480×480px
  - `posts/` — post/social images (e.g. `post-slug.jpg`), recommended 1200×628px

Use web-friendly formats (`jpg`, `png`, `webp`) and optimize for web delivery.
